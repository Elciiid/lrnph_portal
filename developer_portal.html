1. Project Overview

The Main Portal (index.php) is the primary user interface for employees. It provides a central hub for accessing internal applications, viewing company news, and managing schedules.

Core Responsibilities

  • Dynamic App Menu: Renders applications based on user permissions.
  • Announcement Banners: Displays headline news with high-impact visuals.
  • Calendar & Schedule: Shows upcoming events and personal schedules.
  • Responsive Navigation: Provides a seamless experience across mobile and desktop.

2. Layout Architecture

The portal uses a 3-column independent scrolling grid on desktop.

Desktop Breakpoint (lg:)

  • Left Column: Fixed "pill" sidebar for main navigation.
  • Center Column: Main content area containing the App Launcher and Headlines.
  • Right Column: Secondary info area for the Calendar and Announcements list.

Component Map

index.php ├── components/sidebar.php # Fixed navigation pill ├── components/header.php # Top bar (mobile menu toggles) ├── components/announcements.php # Center headline + Right side list └── components/app_grid.php # Dynamic application launcher

3. Frontend Interactivity

Main portal interactivity is powered by Vanilla JS and Tailwind's off-canvas patterns.

Off-Canvas Drawers (Mobile)

On mobile, the sidebars transform into drawers. The #globalBackdrop element manages click-outside logic to close active panels. 

                graph LR
                    MenuBtn[Menu Button] -->|Toggle| Sidebar[Left Sidebar Drawer]
                    InfoBtn[Info Button] -->|Toggle| RightPanel[Right Sidebar Drawer]
                    Backdrop[Global Backdrop] -->|Click| CloseAll[Close All Drawers]

4. Visual Effects (Three.js)

The portal features a particle background effect for the headline banner on desktop devices.

Technical Implementation

Uses THREE.BufferGeometry and THREE.Points for performance. Only initialized if window.innerWidth >= 1024.

Performance Management

Animations are throttled and automatically paused when the banner is not in view to reduce GPU load.
