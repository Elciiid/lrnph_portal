1. Project Overview

The Employee Portal serves as a central hub for employees to access internal applications, view company announcements, and manage their schedules. It features a modern, responsive design that adapts seamlessly from desktop to mobile devices.

Key Features

  • App Launcher: A grid of internal applications with search and "favorite" functionality.
  • Announcements: Company news displaying headlines and detailed updates.
  • Responsive Layout: Optimized for desktop (3-column layout) and mobile (toggleable off-canvas drawers).
  • Admin Panel: A dashboard for managing content, users, and settings.
  • Visual Effects: Three.js particle background for headlines (Desktop only).

2. Technology Stack

Backend

  • Native PHP (No framework)
  • SQL Server (MSSQL)
  • sqlsrv Driver for PHP

Frontend

  • Tailwind CSS (Utility-first styling via CDN)
  • Vanilla JavaScript (No jQuery dependency)
  • Three.js (Used for the particle background effect)

3. Architecture & Process Flows

System Architecture

The application follows a simple monolithic pattern where PHP handles both business logic and presentation rendering.

                    graph TD

                        Components -->|Fetch data| DB
                        DB -->|Return data| Components

Admin Role Checking Flow

A typical check for admin page access (admin.php):

                    flowchart LR

Routing Note

The admin panel (admin.php) uses query-parameter based routing (?page=). The content displayed is determined by including the matching physical file from the components/ directory.

6. Common Code Snippets

Database Connection (includes/db.php)

Connecting to the MSSQL Database using the sqlsrv driver array configuration.

<?php $serverName = "YOUR_SERVER_IP"; $connectionOptions = [ "Database" => "YOUR_DB_NAME", "Uid" => "YOUR_USERNAME", "PWD" => "YOUR_PASSWORD" ]; // Establishes the connection $conn = sqlsrv_connect($serverName, $connectionOptions); if ($conn === false) { die(print_r(sqlsrv_errors(), true)); } ?>

Basic Query Example

Example of standard parameterless query utilizing sqlsrv_query and fetching.

$sql = "SELECT title, description, image_url FROM portal_announcements WHERE is_active = 1 AND type = 'headline' ORDER BY created_at DESC"; $stmt = sqlsrv_query($conn, $sql); if ($stmt === false) { die(print_r(sqlsrv_errors(), true)); } while ($row = sqlsrv_fetch_array($stmt, SQLSRV_FETCH_ASSOC)) { echo "<h2>" . htmlspecialchars($row['title']) . "</h2>"; }

Dynamic Admin Routing (`admin.php`)

The application serves its administrative views via simple switch statement routing.

<?php // Retrieve requested page or fallback to dashboard $page = $_GET['page'] ?? 'dashboard'; switch ($page) { case 'dashboard': include 'components/dashboard.php'; break; case 'announcements': include 'components/announcements.php'; break; // Add new cases here for new pages default: include 'components/404.php'; break; } ?>
